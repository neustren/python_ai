<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/neustren/python_ai.git
   ```
2. Install python packages
   ```sh
   python -m venv venv
   source venv/bin/activate # GNU/Linux
   pip install -r requirements.txt
   ```
